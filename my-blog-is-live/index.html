My blog is live

September 27, 2017 | By: Mark Anthony Uy

Finaly my blog is up and running.

It was a very challenging week for me as I decided to create a blog again this time using static generator. I've been blogging since 2010 using blogspot and wordpress. Although all those blogs are now outdated and removed I don't want to abandom my passion for writing.

Ever since I saw jekyll I wanted to try it but for some reason didn't happen. Maybe because of the fact that it was built on Ruby and I have no knowledge of it. Good thing I found a great tutorial by LevelUp Tuts on Gatsbyjs. It was a walkthrough, easy to follow and great for beginners. I was intimidated at first because of the technologies used in Gatsbyjs. Besides Reactjs they used Graphql although new I heard a lot of good feedbacks about it. But since I already have built web apps with react I decided to choose Gatsbyjs.

All in all I'm very happy on what I've accomplished. I will be able to create a blog post using markdown and host it on github which is the best part for me.

Finally, I'd like to thank to the people behind all these great tools and technologies I mentioned above!
